# Zhou Dynasty
An adventure game created after Gary Zhou

### Controls
```
Shoot - Attack the enemies
Defend - Defend from the enemies
Run - Run from the enemies 
```

### Classes
```
Archer      ❤ HP: 20 | ❁ Dmg: 20
Mage        ❤ HP: 15 | ❁ Dmg: 30  
```
## Mobs
```
Skeleton          ❤ HP: 30 | ❁ Dmg: 2
Wither Skeleton   ❤ HP: 50 | ❁ Dmg: 3
Wither            ❤ HP: ???| ❁ Dmg: ???
```
